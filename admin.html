📋 How to Publish

🚀 GitHub Pages Setup (one time)

1

Go to github.com, create a free account, then create a new repository named my-story-corner (set it Public)

2

Upload index.html, admin.html, and stories.json to the repo root

3

Create a folder called stories/ and upload your PDF files there

4

Go to repo Settings → Pages, set source to main branch. Your site will be live at:
https://YOUR-USERNAME.github.io/my-story-corner

5

Each time you add a story here, download the new stories.json and re-upload it to GitHub along with the new PDF file in the stories/ folder
