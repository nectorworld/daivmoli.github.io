# 📚 My Story Corner — Setup Guide

## Files in this folder
```
index.html      ← Kids' reading page (share this URL with everyone)
admin.html      ← Your private page to manage stories
stories.json    ← List of stories (you update & re-upload this)
stories/        ← Folder where all PDF files go
```

---

## 🚀 One-time GitHub Setup (free hosting)

1. Go to **github.com** → Sign up for a free account
2. Click **"New repository"** → name it `my-story-corner` → set to **Public** → Create
3. Upload these files to the repo:
   - `index.html`
   - `admin.html`
   - `stories.json`
4. Create a folder called `stories` (upload any dummy file inside it to create it)
5. Go to **Settings → Pages** → Source: `Deploy from branch` → Branch: `main` → Save
6. Your site is live at: `https://YOUR-USERNAME.github.io/my-story-corner`

---

## 📱 Adding a Story (daily workflow)

1. Open `admin.html` in your browser
2. Enter password: **jaipur10190**
3. Fill in the title, pick an emoji, choose the PDF file
4. Tap **"Add to Story List"**
5. Tap **"Download stories.json"** — save this file
6. Go to your GitHub repo:
   - Upload the new PDF into the `stories/` folder
   - Replace the old `stories.json` with the new one
7. Done! Kids see the new story within seconds ✅
